Iterator: Iterator란 자바의 컬렉션(Collection)에 저장되어 있는 요소들을 순회하는 인터페이스이다.

- 정의방법은 Iterator<T> iterator = Collection.iterator();
- 메서드는 다음 요소가 있는지 판단하는 hasNext(), 다음 요소를 가져오는 next(), 가져온 요소를 삭제하는 remove()
